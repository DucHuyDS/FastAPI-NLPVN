# FastAPI-NLPVN

## How to run

1.  Install packages
  ```bash
  pip install -r requirements.txt
  ```
2.  Create .env file with content:
   ```bash
   cookie=<YOUR_COOKIE_FACEBOOK>
   ```
3.  Run server
   ```bash
   python app/main.py
   ```
